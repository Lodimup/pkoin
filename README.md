# pkoin
Pakin's shit coin finder - a program to find profitable shit coins.  

This program is name in an hornor of my friend, Pakin who came up with this strategy.  

**Key assumptions**

Shit coins are more volatile than the market but has less downside since people tends to HODL them when the market is bear.

**How to install**  
Install Python 3 via Anaconda: https://www.anaconda.com/products/individual  
clone this project  
```git clone https://github.com/Lodimup/pkoin.git```  
```pip install -r requirements.txt```  
run via   
```python main.py```  
or   
```python3 main.py```  

What you do with information gained from this program is at your own risk. This is not a financial advice. Please trade responsibly.
